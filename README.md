# RKP Frontend Next.js App

## Run

First, run the development server:

```bash
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Enviroment config

_.env.local file_

```js
API_URL=http://localhost:1337 // url to backend
```
